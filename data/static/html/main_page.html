Школа::Кода

Научиться программировать может каждый!

Образовательный проект "Школа :: Кода" работает с 2006 г. при кафедре математического обеспечения и применения ЭВМ (МОП ЭВМ) Института компьютерных технологий и информационной безопасности (ИКТИБ) Южного федерального университета.

Занятия проходят с октября по май, 2-3 раза в неделю, как правило в вечернее время, их проводят преподаватели кафедры МОП ЭВМ, специалисты ИТ-компаний, магистранты ИКТИБ.

Обучающие программы

  • Языки визуального программирования и учебные языки (Scratch) - для учащихся 4-8 классов, не знакомых с информатикой.
  • Языки высокого уровня (Python, Java, C/C++) - для учащихся 7-11 классов, имеющих базовые знания по информатике.
  • Подготовка к олимпиадам, объектно-ориентированное программирование, компьютерная графика, веб-программирование - для тех, кто уже имеет серьезную базовую подготовку по информатике и программированию.
